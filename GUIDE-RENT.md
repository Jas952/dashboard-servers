# Как искать серверы на Vast под Pearl AlphaPool

Краткая инструкция: какие GPU смотреть, как считать экономику и что проверять после аренды. Ориентиры взяты с рабочего флота (~$0.17–0.21 за 100 TH на пуле).

---

## 1. Цель по экономике

| Метрика | Норма | На грани | Не брать |
|--------|--------|----------|----------|
| **$/100TH** (оценка) | **≤ $0.20** | $0.20–0.25 | **> $0.25** |
| Хороший основной флот (pearl) | ~**$0.15–0.18**/100TH | — | — |
| Весь флот в среднем | ~**$0.20–0.21**/100TH LIVE | — | — |

### Формула

```text
$/100TH = (цена $/час на Vast) ÷ (ожидаемый хешрейт в TH/s) × 100
```

Хешрейт — **оценка по типу GPU** для сравнения офферов (не гарантия пула):

| GPU | TH/s на 1 GPU (ориентир) |
|-----|---------------------------|
| RTX 5090 | ~310 |
| RTX 5080 | ~160 |
| RTX 4090 | ~145 |
| RTX 4080 | ~120 |
| RTX 3090 | ~100 |
| RTX 3080 Ti | ~75 |
| RTX 3080 | ~65 |
| RTX 3070 | ~45 |

**Пример:** 2×3090, $0.29/ч → ожид. ~200 TH → $0.29 ÷ 200 × 100 = **$0.145/100TH** — хорошо.

---

## 2. Какие GPU смотреть

**В приоритете:**

- **RTX 3090** — лучший баланс цена/хеш, много офферов 1–4 GPU
- **RTX 5080, RTX 5090** — много TH, но на рынке часто дорого; 5090 только если **≤ ~$0.22/100TH**
- **RTX 4080** — редко бывает дёшево; чаще > $0.25/100TH
- **RTX 3080 Ti** — для тестов, если **≤ ~$0.20/100TH**

**Обычно не целимся в:**

- H100 и датацентровые «премиум» GPU — для Pearl часто **$0.40+/100TH**
- Мелкие карты (4060, 4070 и т.д.) — только если явно **≤ $0.20/100TH**
- Пакеты **8×5090** за $5–6/ч — почти всегда **> $0.24/100TH**

---

## 3. Фильтры на Vast.ai

При поиске офферов:

- **Тип:** on-demand, статус **rentable**
- **Доля GPU:** **≥ 99%** (полная карта)
- **Reliability:** **≥ ~76%** (лучше **90%+**)
- **Число GPU:** чаще **1–4**; 8 GPU — только если посчитали $/100TH вручную
- **SSH / direct port:** желательно (проверка и перезапуск майнера)

**Шаблон:** Pearl AlphaPool Miner (актуальная версия onstart, например v1.6). Использовать **свой** кошелёк и **свой** `template_hash` / rent-ссылку из аккаунта Vast.

---

## 4. География

**Чаще нормально:** US (TX, GA, PA, NV), Canada (Quebec, Ontario, BC), EU (FR, PL, CZ, ES, IT, LV), Romania, Belgium.

**Осторожно** (выше риск ошибок Docker / 0 TH на пуле): часть UA, JP, QA, MX, крупные UK-пакеты. Не запрет, но после аренды обязательно проверить pool LIVE; при сбое — destroy и не брать тот `machine_id` снова (blacklist).

**Обычно пропускаем:** RU, CN, HK, NL, Iceland, Malaysia, Brazil и похожие регионы (цена, стабильность, пинг к пулу).

---

## 5. Пошаговый поиск

1. Открыть [cloud.vast.ai](https://cloud.vast.ai/) → поиск офферов.
2. Отфильтровать нужные GPU (3090 / 5080 / 5090).
3. Для каждого оффера посчитать **$/100TH** (таблица TH выше).
4. Брать только **≤ $0.20/100TH** (тестовые хосты — до **$0.22**).
5. Задать **label** (`pearl-3090`, `test` и т.д.) для учёта флота.
6. Через **10–30 мин** проверить **pool LIVE** (не только `gpu_util` в Vast).
7. Нет хеша на пуле **> 15–20 мин** → destroy, хост в blacklist.

## 7. Бюджет и роли

- **Основной флот (pearl-*):** цель **~$0.15–0.18/100TH**, стабильные хосты.
- **Тест (label `test`):** новые офферы с рынка; разумный потолок **~$5/ч** на всю группу test. Часть аренд отвалится по Docker — это нормально; мёртвые инстансы сразу удалять.

---

## 8. Проверка после аренды

| Где | Что смотреть |
|-----|----------------|
| **Пул** | LIVE hashrate, воркер `vast-<instance_id>-...` |
| **SSH** (если доступен) | `/var/log/alpha-miner.log`, строки `hashrate_th_s=` |
| **Vast** | статус `running`, в логе нет `CDI`, `failed to create task`, `invalid runtime nvidia` |

**Ориентиры LIVE (не в первые 2–3 минуты):**

- 3090: **> ~50 TH**
- 5080: **> ~120 TH**
- 5090: **> ~200 TH**

Первые минуты хеш ниже — нормально; смотреть также **1h** на пуле.

---

## 9. Типичные ошибки

- Смотреть только **$/GPU/ч** без пересчёта в **$/100TH**
- Брать 5090/4080 «потому что топ», не считая экономику
- Не удалять инстансы с **0 TH на пуле** — платёж без майнинга
- Доверять Vast **gpu_util 100%** при **0 TH на пуле**

---

## 10. Шпаргалка одной строкой

> **3090/5080, полная GPU, rel ≥ 90%, $/100TH ≤ $0.20, US/CA/EU, шаблон Pearl, через 15 мин pool LIVE — нет хеша = destroy.**

## Скрипты в этом каталоге (опционально)

При наличии `VAST_API_KEY` в окружении:

```bash
python scan-fleet-gpus.py      # 3090/5080/5090/4080, ≤ $0.22/100TH
python scan-series-3050.py     # серии 30/40/50
python scan-target-4gpu.py     # только 3090/4080/5080/5090
python fleet-all-status.py     # LIVE пул vs все инстансы
python check-test-batch.py     # статус label=test
```

Файл `blacklist.json` — `machine_id` / офферы с Docker/CDI/GPU-сбоями, чтобы не арендовать повторно.
